Paso a paso prueba AXEDE

1. Clonar el repositorio 
2. Instalar python y pip
3. Importar  la base de datos MySQL (Tener MySQL 8.0 o superior)
4. Abrir el proyecto en la terminal
5. Iniciar el entorno virtual con el siguiente comando “Source ven/bin/Activate”
6. Instalar los requerimientos con el comando “pip install -r requirements.txt”
7. Entrar al directorio myproject con el comando “cd myproject”
8. Correr el servidor con el comando “python3 manage.py runserver”
9. Abrir el enlace que se genere en la consola, generalmente es “http://127.0.0.1:8000/“
